# MachnetQuery
This is jQuery like lightweight implementation in vanilla js. Feel free to add functions and use

## Usage Example

Your html
```
<div class="wrapper">
  <h1>This is a headline</h1>
</div>
```

### select element

```js
MQ('.wrapper');
```

### hide element
```js
MQ('.wrapper').hide();
```

### show element
```js
MQ('.wrapper').show();
```
